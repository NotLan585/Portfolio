My Projects

Automation Data Graphing

This project was something that didn't need to be done, but I wanted to learn more about showing data in a easy to read way to whenever automation is ran. For this I created a slackbot and setup the appropriate token/webhooks to post the message. When running the automation for the suite, I setup a autouse fixture to capture data at the end of the test("passed/failed/skipped"/"Length of test in second") and used matplotlib and plotted the data and subplots and used slack_sdk to upload the file and inlay it with the main suite results. For the suite results(For the nightly automation) when running the nightly I `| tee results.txt` all the output to a .txt file and set a `after_script` in the pipeline to parse the file and use Slacks BlockKitBuilder to setup the payload to send the data of the outcome of the nightly automation.

I think the next thing I'll do for this project is more data such as amount of clicks or actions performed and continue to learn more about Jupyter
